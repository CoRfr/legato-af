Legato

This version of Legato supports the following features:

  • Modem services APIs: SMS, SIM, Data connection, voice control, and positioning APIs.
  • C runtime library
  • Application supervisor, inter-process communication (IPC)
  • Basic sandbox
  • Development framework
  • Component generation tools
  • Code generation and build scripts
  • Demos, tutorials, and documentation

Linux BSP

The following BSP features are validated and supported on boards where the corresponding hardware features exist:

General Serial Bus Interface

  • UART
  • I2C Master
  • SPI Master

USB Controller

  • High Speed USB Host Controller
  • High Speed USB Device Controller
  • High Speed OTG Host Controller
  • HSIC USB Host Controller
  • USB Devices(USB Host Controller functions)
  • USB Audio
  • USB ACM
  • USB Ethernet
  • USB hub
  • USB Storage

USB Gadget (USB Device Controller functions)

  • CDC-ACM
  • CDC-ECM
  • Storage
  • Built-in Qualcomm 3G modem(Proprietary code is needed)

Network

  • SMSC95XX HSIC Ethernet Controller

Storage

  • NAND Flash: flexible static memory controller, FSMC
  • NAND Flash: Micron NAND Flash MT29F2G08, 2Gb
  • SD/MMC: On-Chip SDCC core

Misc Devices

  • GPIO: On-Chip general purpose I/O peripheral

Debug Features

  • Gdbserver
  • Lttng
  • Strace
  • Kgdboc
  • Valgrind
